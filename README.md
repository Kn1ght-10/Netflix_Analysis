# 🎬 Netflix Content Analysis

An exploratory data analysis of Netflix's catalog of movies and TV shows, visualized with Tableau.

## Overview
This project analyzes Netflix's title catalog to surface trends in content type, release year, genre, country of origin, and ratings over time.

## Contents
- `netflix_titles.csv.zip` — dataset of Netflix movies and TV shows
- `Netflix_Analysis.twbx` — interactive Tableau packaged workbook

## Tools Used
- Tableau
- CSV / Excel

## How to View
Unzip `netflix_titles.csv.zip` for the raw data, or open `Netflix_Analysis.twbx` in Tableau Desktop or Tableau Public to explore the interactive dashboard.
