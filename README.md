**An interactive Tableau exploration of Netflix's full catalog — what the platform's content strategy looks like when you visualize every title.**

## 🔎 What this project answers

- **Movies vs. TV shows** — how the content mix has shifted over time
- **Genre trends** — which categories dominate and which are growing
- **Geography** — country-of-origin patterns in Netflix's catalog
- **Ratings** — maturity-rating distribution across content types
- **Release timing** — catalog age vs. date added to the platform

## 📁 Contents

| File | Description |
| --- | --- |
| `Netflix_Analysis.twbx` | Interactive Tableau packaged workbook |
| `netflix_titles.csv.zip` | Raw dataset of Netflix movies and TV shows |

## ▶️ How to view

Open `Netflix_Analysis.twbx` in **Tableau Desktop** or the free **[Tableau Public](https://public.tableau.com/)** app, or unzip the CSV to explore the raw data.
